¿En qué espacio vive la regresión?

Una visualización progresiva para relacionar \(\mathbb{R}\), \(\mathbb{R}^2\), \(\mathbb{R}^3\) y \(\mathbb{R}^n\).

← Regresar semana II

⚙ Dimensión del espacio

Idea:
Cada variable agrega una dimensión.

Una observación con \(d\) variables vive en: \[ \mathbb{R}^d \]
\(\mathbb{R}\): una variable, un eje.
\(\mathbb{R}^2\): regresión simple, puntos \((X,Y)\).
\(\mathbb{R}^3\): regresión múltiple con dos predictores, puntos \((X_1,X_2,Y)\).
\(\mathbb{R}^n\): más dimensiones, ya no se puede visualizar directamente.

Visualización conceptual

Espacio \(\mathbb{R}^2\)
Representación Plano 2D
Modelo visible Recta
Idea central: la regresión lineal simple se puede visualizar en \(\mathbb{R}^2\), porque cada observación tiene dos componentes: \((X,Y)\). La regresión múltiple con dos predictores se puede visualizar en \(\mathbb{R}^3\), porque cada observación tiene tres componentes: \((X_1,X_2,Y)\). Cuando el número de variables aumenta, el espacio sigue existiendo matemáticamente, pero ya no podemos dibujarlo. Por eso usamos medidas numéricas como \(R^2\), \(\hat{R}^2\), MSE o RMSE para evaluar el ajuste.
