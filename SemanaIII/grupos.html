Variabilidad entre grupos

Simulador introductorio para observar cómo se separan las medias de grupo respecto a la media general.

⚙ Parámetros

Media general: \[ \bar{Y}=\frac{1}{N}\sum_{j=1}^{k}\sum_{i=1}^{n_j}Y_{ij} \]
Variabilidad entre grupos: \[ SC_{\text{entre}} = \sum_{j=1}^{k} n_j(\bar{Y}_j-\bar{Y})^2 \]
