Componentes utilizados en la simulación

Modelo poblacional

\[ Y_i=\beta_0+\beta_1X_i+\varepsilon_i \]
\(\beta_0\) Intercepto poblacional
\(\beta_1\) Pendiente poblacional
\(X_i\) Variable explicativa
\(Y_i\) Variable respuesta
\(\varepsilon_i\) Error aleatorio

Componente aleatoria

\[ \varepsilon_i\sim N(0,\sigma^2) \]
\(\sigma\) pequeño

Menor dispersión alrededor de la función de regresión.

\(\sigma\) grande

Mayor variabilidad y menor precisión inferencial.

El parámetro \(\sigma\) controla la variabilidad de las observaciones alrededor de la recta poblacional.

Asociación lineal

\[ r= \frac{\sum (X_i-\bar X)(Y_i-\bar Y)} {\sqrt{\sum (X_i-\bar X)^2 \sum (Y_i-\bar Y)^2}} \]
\(r\)

Intensidad y dirección de la asociación lineal.

\(R^2\)

Proporción de variabilidad explicada por el modelo.

Inferencia estadística

\[ H_0:\beta_1=0 \] \[ H_1:\beta_1\neq0 \]
\(p<0.05\)

Evidencia estadística de relación lineal.

\(p\ge0.05\)

No existe evidencia suficiente para rechazar \(H_0\).

Predicción

\[ \hat Y_0= \hat\beta_0+ \hat\beta_1x_0 \]
IC 95%

Intervalo para la respuesta media \(E(Y|X=x_0)\).

IP 95%

Intervalo para una observación futura individual.

El intervalo de predicción siempre es más amplio que el intervalo de confianza para la media.
