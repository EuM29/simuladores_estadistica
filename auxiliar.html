Atajos rápidos para Excel

Combinaciones de teclas útiles para trabajar más rápido con datos, fórmulas, tablas y análisis.

← Regresar a Semana I

01. Navegación rápida

↔️

Ir al final de una tabla

Moverse rápidamente hasta el último dato en una dirección.

Ctrl+↑ ↓ ← →
🏁

Ir a la celda A1

Regresa al inicio de la hoja de cálculo.

Ctrl+Inicio
📍

Ir a una celda específica

Permite escribir una referencia como B25 o Hoja2!A1.

F5

02. Selección de datos

🔲

Seleccionar región actual

Selecciona rápidamente toda la tabla o bloque de datos.

Ctrl+*
📋

Seleccionar columna completa

Útil para aplicar formato o revisar una variable.

Ctrl+Espacio
↕️

Seleccionar fila completa

Selecciona toda la fila activa.

Shift+Espacio

03. Fórmulas y edición

🧮

Insertar función

Abre el asistente para buscar funciones como promedio, correlación o pendiente.

Shift+F3
✏️

Editar celda activa

Permite modificar la fórmula o contenido sin usar doble clic.

F2
🔁

Repetir última acción

Aplica nuevamente el último formato o acción realizada.

F4
💲

Fijar referencias

Cambia entre A1, $A$1, A$1 y $A1 dentro de una fórmula.

F4

04. Formato y análisis

📊

Crear gráfico rápido

Genera un gráfico con los datos seleccionados.

Alt+F1
🧾

Crear tabla

Convierte un rango en tabla con filtros y formato automático.

Ctrl+T
🔍

Activar filtros

Agrega o quita filtros en la fila de encabezados.

Ctrl+Shift+L
📌

Formato de celdas

Abre la ventana para número, alineación, bordes y relleno.

Ctrl+1

05. Productividad general

💾

Guardar archivo

Guarda rápidamente los cambios realizados.

Ctrl+G
↩️

Deshacer

Revierte la última acción realizada.

Ctrl+Z
🔎

Buscar

Busca valores, texto o fórmulas dentro de la hoja.

Ctrl+B
🖨️

Imprimir

Abre la vista de impresión del libro.

Ctrl+P
Sugerencia didáctica: practique estos atajos mientras trabaja con bases de datos pequeñas. La velocidad aumenta cuando se combinan navegación, selección y fórmulas sin depender tanto del mouse.
