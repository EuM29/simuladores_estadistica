Atajos rápidos para Excel

Combinaciones de teclas útiles para trabajar más rápido con datos, fórmulas, tablas y análisis.

← Regresar a Semana I

01. Navegación rápida

↕️

Moverse al final de los datos

Salta directamente al último dato de una columna o fila.

Ctrl + ↑ ↓ ← →
📍

Ir a una celda específica

Permite escribir una referencia como B250 o A1000.

F5
🪟

Cambiar entre ventanas

Cambia rápidamente entre Excel, Word, navegador y otros programas.

Alt + Tab

02. Selección masiva de datos

⬇️

Seleccionar hacia abajo

Selecciona todas las celdas hasta el último dato.

Ctrl + Shift + ↓
⬆️

Seleccionar hacia arriba

Ctrl + Shift + ↑
➡️

Seleccionar hacia la derecha

Ctrl + Shift + →
⬅️

Seleccionar hacia la izquierda

Ctrl + Shift + ←
📋

Seleccionar columna completa

Ctrl + Espacio
📑

Seleccionar fila completa

Shift + Espacio
🔲

Seleccionar toda la región

Selecciona automáticamente toda la tabla de datos.

Ctrl + Shift + Espacio

03. Productividad para análisis

🔍

Activar filtros

Ctrl + Shift + L
💾

Guardar

Ctrl + G
↩️

Deshacer

Ctrl + Z
↪️

Rehacer

Ctrl + Y
🔎

Buscar

Ctrl + B
Consejo: En análisis estadístico es común trabajar con miles de registros. Los atajos de navegación y selección suelen ahorrar más tiempo que cualquier otra combinación de teclas.
