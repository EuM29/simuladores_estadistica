1. Modelo poblacional

\[ Y_i=\beta_0+\beta_1X_i+\varepsilon_i \]

Representa la relación teórica entre la variable explicativa \(X\) y la variable respuesta \(Y\).

2. Modelo estimado

\[ \hat{Y}_i=\hat{\beta}_0+\hat{\beta}_1X_i \]

Es la recta ajustada a partir de la muestra.

Excel pendiente: Español: =PENDIENTE(rango_y; rango_x) English: =SLOPE(y_range, x_range) Excel intercepto: Español: =INTERSECCION.EJE(rango_y; rango_x) English: =INTERCEPT(y_range, x_range)

3. Cantidades auxiliares

\[ S_{xx}=\sum (X_i-\bar X)^2 \]
\[ S_{xy}=\sum (X_i-\bar X)(Y_i-\bar Y) \]

Estas cantidades se usan para calcular la pendiente, el error estándar y los intervalos.

4. Pendiente estimada

\[ \hat{\beta}_1=\frac{S_{xy}}{S_{xx}} \]

Indica cuánto cambia \(Y\), en promedio, cuando \(X\) aumenta una unidad.

5. Intercepto estimado

\[ \hat{\beta}_0=\bar Y-\hat{\beta}_1\bar X \]

Es el valor estimado de \(Y\) cuando \(X=0\).

10. Error estándar del modelo

\[ s=\sqrt{\frac{SSE}{n-2}} \]

Mide la dispersión promedio de los datos alrededor de la recta ajustada.

11. Errores estándar de los coeficientes

\[ SE(\hat{\beta}_1)=\frac{s}{\sqrt{S_{xx}}} \]
\[ SE(\hat{\beta}_0)= s\sqrt{ \frac{1}{n} + \frac{\bar X^2}{S_{xx}} } \]

Se usan para construir intervalos de confianza y pruebas de hipótesis.

12. Prueba de hipótesis para la pendiente

\[ H_0:\beta_1=0 \qquad H_1:\beta_1\neq0 \]
\[ t= \frac{\hat{\beta}_1} {\frac{s}{\sqrt{S_{xx}}}} = \frac{\hat{\beta}_1\sqrt{S_{xx}}}{s} \]

Evalúa si existe evidencia estadística de una relación lineal entre \(X\) y \(Y\).

13. Intervalo de confianza para \(\beta_1\)

\[ \hat{\beta}_1 \pm t_{\alpha/2,n-2} \left( \frac{s}{\sqrt{S_{xx}}} \right) \]

14. Intervalo de confianza para \(\beta_0\)

\[ \hat{\beta}_0 \pm t_{\alpha/2,n-2} \left[ s\sqrt{ \frac{1}{n} + \frac{\bar X^2}{S_{xx}} } \right] \]
