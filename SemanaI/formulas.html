Resumen de fórmulas y funciones de Excel en español e inglés para apoyar los cálculos de la Semana I.

1. Modelo poblacional

\[ Y_i=\beta_0+\beta_1X_i+\varepsilon_i \]

Representa la relación teórica entre la variable explicativa \(X\) y la variable respuesta \(Y\).

2. Modelo estimado

\[ \hat{Y}_i=\hat{\beta}_0+\hat{\beta}_1X_i \]

Es la recta ajustada a partir de la muestra.

Excel pendiente: Español: =PENDIENTE(rango_y; rango_x) English: =SLOPE(y_range, x_range) Excel intercepto: Español: =INTERSECCION.EJE(rango_y; rango_x) English: =INTERCEPT(y_range, x_range)

3. Pendiente estimada

\[ \hat{\beta}_1= \frac{\sum (X_i-\bar X)(Y_i-\bar Y)} {\sum (X_i-\bar X)^2} \]

Indica cuánto cambia \(Y\), en promedio, cuando \(X\) aumenta una unidad.

4. Intercepto estimado

\[ \hat{\beta}_0=\bar Y-\hat{\beta}_1\bar X \]

Es el valor estimado de \(Y\) cuando \(X=0\).

5. Correlación de Pearson

\[ r= \frac{\sum (X_i-\bar X)(Y_i-\bar Y)} {\sqrt{\sum (X_i-\bar X)^2\sum (Y_i-\bar Y)^2}} \]

Mide la dirección y fuerza de la relación lineal entre dos variables.

Excel: Español: =COEF.DE.CORREL(rango_x; rango_y) English: =CORREL(x_range, y_range)

6. Coeficiente de determinación

\[ R^2=r^2 \]

Representa la proporción de variabilidad de \(Y\) explicada por el modelo lineal.

Excel: Español: =COEFICIENTE.R2(rango_y; rango_x) English: =RSQ(y_range, x_range)

7. Residuo

\[ e_i=Y_i-\hat{Y}_i \]

Es la diferencia entre el valor observado y el valor estimado por la recta.

8. Suma de cuadrados de los residuos

\[ SSE=\sum e_i^2=\sum(Y_i-\hat{Y}_i)^2 \]

Mide el error total del modelo. Mínimos cuadrados busca hacer este valor lo más pequeño posible.

9. Error estándar del modelo

\[ s=\sqrt{\frac{SSE}{n-2}} \]

Mide la dispersión promedio de los datos alrededor de la recta ajustada.

Excel: Español: =ERROR.TIPICO.XY(rango_y; rango_x) English: =STEYX(y_range, x_range)

10. Prueba de hipótesis para la pendiente

\[ H_0:\beta_1=0 \] \[ H_1:\beta_1\neq0 \]
\[ t=\frac{\hat{\beta}_1}{SE(\hat{\beta}_1)} \]

Evalúa si existe evidencia estadística de una relación lineal entre \(X\) y \(Y\).

p-valor: Español: =DISTR.T.2C(abs(t); grados_libertad) English: =T.DIST.2T(ABS(t), degrees_freedom)

11. Intervalo de confianza para un coeficiente

\[ \hat{\beta}_j \pm t_{\alpha/2,n-2}SE(\hat{\beta}_j) \]

Se utiliza para estimar un rango plausible para \(\beta_0\) o \(\beta_1\).

Valor crítico t: Español: =INV.T.2C(0.05; n-2) English: =T.INV.2T(0.05, n-2)

12. Predicción puntual

\[ \hat{Y}_0= \hat{\beta}_0+\hat{\beta}_1X_0 \]

Es el valor estimado de \(Y\) para un valor específico \(X_0\).

13. Intervalo para la media

\[ \hat{Y}_0 \pm t_{\alpha/2,n-2} s\sqrt{ \frac{1}{n}+ \frac{(X_0-\bar X)^2}{\sum(X_i-\bar X)^2} } \]

Estima la respuesta media esperada para \(X=X_0\).

14. Intervalo de predicción

\[ \hat{Y}_0 \pm t_{\alpha/2,n-2} s\sqrt{ 1+\frac{1}{n}+ \frac{(X_0-\bar X)^2}{\sum(X_i-\bar X)^2} } \]

Estima el rango probable para una observación individual futura.
